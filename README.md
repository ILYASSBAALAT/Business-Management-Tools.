📊 Business Suite Morocco (BSM) v1.0
Automated Financial Management for the Moroccan Market
🚀 Overview
Business Suite Morocco est une solution d'automatisation développée pour simplifier la gestion financière opérationnelle. Conçue initialement pour répondre aux besoins de précision lors de mon analyse de rentabilité au sein du Fairmont Tazi Palace, cette application permet de calculer instantanément le profit net et la TVA en respectant les standards comptables marocains.
✨ Key Features
Net Profit Calculation: Calcul précis basé sur les revenus bruts et les charges d'exploitation.
Moroccan Tax Engine: Gestion automatisée de la TVA (standard & réduit).
Operational Agility: Interface légère en ligne de commande pour une exécution rapide sur le terrain.
Data Integrity: Réduction de 95% des erreurs de calcul par rapport aux saisies manuelles sur papier.
🛠️ Tech Stack
Language: Python 3
Environment: Cross-platform (Replit, VS Code, Terminal)
Logic: Financial Management Algorithms (TSGE Standards)
